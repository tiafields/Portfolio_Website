Shoe Size Conversion Application in Python

Role: Full Stack Developer and Tkinter GUI Designer
Source Code:

When traveling abroad in London with a friend. She was in need of a new pair of shoes. Although it is easy to search online for a shoe conversion the idea struck me and I said “What if there was an application for this?”

My very first coding project created as a final for a software development class out of my certification for Ivy Tech I created a web application in Python Tkinter to convert American Shoe Sizes to European Shoe Sizes and vise versa. This would include Male and Female sizes as the shoe sizes for gender vary.

With this application the user is able to enter the shoe size in the text field and press the “convert to” button and it will output the other countries shoe size below.

It will then store all of the data the user has typed into an external window titled “Your Conversions” You can then close this window and clear your entries and start the process again.
